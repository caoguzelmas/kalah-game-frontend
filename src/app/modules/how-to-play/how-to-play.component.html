1. Creating a New Game
You can create a new game via "Create New Game" section with configuring variations.
1.1. First Move Variation
You can decide who starts the game via enabling/disabling First Move Variation.
1.2. Game Flow Direction Variation
You can decide which way the game flows via enabling/disabling Game Flow Direction. Standard game flows counterclockwise.
1.3. Empty Capture Rule Variation
If the last sown seed lands in an empty house owned by the player, even if the opposite house is empty, the last seed is captured and placed into the player's store.
1.4. Remaining Stone(s) Insertion Variation
Remaining Stone(s) Insertion Variation enables/disables the rule does not count the remaining seeds as part of the opponent's score at the end of the game.
2. Active Game
You can find out the active game under this section.
3. All Games
You can find all the finished/unfinished games. You can also continue an unfinished game via clicking "Continue to Play" button.
